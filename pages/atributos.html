No Atributo Definición
1 Id proporciona un identificador único en el ámbito de todo el documento para un elemento.
2 Class proporciona una manera de clasificar elementos similares.
3 Style El atributo de estilo se puede utilizar en cualquier elemento de HTML
4 Title se suele usar para unir explicación subtextual a un elemento.
5 Align indica como va a encontrarse alineada la tabla respecto al contenido del documento en cuestión
6 Bgcolor Este atributo define el color de fondo de la tabla.
7 Border Este atributo entero define el tamaño del cuadro alrededor de la tabla.
8 Frame Este atributo enumerativo define cual lado del cuadro alrededor de la tabla será mostrada.
9 Rules Este atributo enumerativo define donde aparecen las reglas en la tabla, por ejemplo líneas.
10 Width Este atributo define el ancho de una tabla, pudiendo contener una longitud de píxeles o un porcentaje, que representa un porcentaje de anchura del contenedor que la tabla debiera usar.
